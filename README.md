# Bechtel Fitness

Static GitHub Pages site for the Bechtel Fitness workout tracker.

## Files

- `index.html` - main app
- `uploads/BLSY1C21.pdf` - original source PDF
- `screenshots/current.png` - preview screenshot
- `data.js` - exported workout data reference

## Publish

Push this folder to a GitHub repository and enable GitHub Pages from the default branch.
